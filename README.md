# PK Remote

An open-source Google TV / Android TV remote built with SwiftUI.

## Features

- 📺 Automatic Google TV discovery
- 🔐 Secure pairing
- 🎮 Remote controls
- ⌨️ Built-in keyboard
- ▶️ Media controls
- ⭐ STBEmu mode
- 🚫 No ads
- 🆓 Open source
